Projects


Custom Crafts

Custom Crafts is a store application where you can view and purchase custom-designed spaceships/spacecrafts. The app allows users to sign-in and become return customers with profile and purchase history features. Customers may shop for different and varying styles of spacecrafts - performance (i.e. racing), luxury, military, etc. - designed to suit their taste and whet their palates.

As the backend lead on this project, I was responsible for designing, and ultimately implementing, a GraphQL api in Flask for Python with Graphene. I also contributed significantly to parsing filter-state on the client to dynamically create query strings in order to achieve various filter and search effects for the store front.

videoEats

Video Eats is an app that allows users to search for video reviews (and text) for restaurants. Users can sign in, search for restaurants by name or category, leave a reviews, vote on reviews, and even add new restaurants!

As the backend lead for this projects, I built a REST api to support our client application. We used BCrypt to sign and verify JWT tokens for user authentication, custom indexes to ensure user vote integrity and persistence, and even some custom request body parsing to expediate the development of our client search functionality.

worldView

World View is designed for the modern world of news media. For better or worse (almost certainly worse), current news narratives are severely impacted by the perspective of its publisher. As such, World View attempts to offer a more rounded news consumption experience by allowing users to quickly fetch articles based on country of origin in addition to general category or topic keyword. It's the hope of World View that this will allow users to efficiently digest a variety of news sources and perspectives, providing a more objective and accurate represenation of current world affairs.

World View uses newsapi, a 3rd party api that sources from over 50,000 news sources across the globe. The frontend uses React and Redux hooks to manage user selection and application state. A PostgreSQL database and REST api on the backend support the saving of user preferences and saved news stories for later use.
